# Gronur Grocery System

A premium, photography-led gourmet grocery storefront featuring a modern desktop-first UI, light-themed designs, and robust inventory management functionality.

## Features
- **Premium Desktop UI**: Modern, responsive, and spacious layout.
- **Full CRUD Support**: Add, Edit, and Delete products instantly.
- **Smart Filtering**: Advanced searching and sorting for efficient shopping.
- **Secure Authentication**: Split-screen login and registration system.
- **Icon-Driven Interface**: Professional SVG icons used throughout for actions and navigation.
- **Cart System**: Real-time quantity tracking and price calculation.

## Tech Stack
- PHP / MySQL
- Vanilla CSS (Premium Light Theme)
- JavaScript (Desktop Optimized)
- Google Fonts (Outfit)
